# 2. 알고리즘 (의사 코드) (10점)

## 1. `main` 함수의 전체 흐름
시작
    현재 숫자를 0으로 설정
    사용자 이름과 컴퓨터 이름 설정

    "베스킨라빈스 31 게임 시작" 출력

    반복 (현재 숫자가 31이 될 때까지):
        --- 사용자 턴 ---
        사용자에게 1~3 중 하나 입력받기
        입력값이 잘못되었으면 다시 입력받기
        선택한 개수만큼 숫자를 증가시키면서 출력
        만약 증가된 숫자가 31이면:
            "사용자 패배" 출력
            게임 종료

        --- 컴퓨터 턴 ---
        필승 전략(4의 배수 만들기)을 기반으로 컴퓨터가 말할 숫자 개수 결정
        결정된 개수만큼 숫자 증가시키면서 출력
        만약 증가된 숫자가 31이면:
            "컴퓨터 패배" 출력
            게임 종료

    반복 끝
종료


## 2. 핵심 기능 1: 사용자 턴 처리 로직
시작
    사용자에게 "몇 개 말할지" 입력받기 (1~3)

    만약 사용자가 1~3이 아닌 숫자를 입력하면:
        "잘못된 입력" 출력 후 다시 입력받기

    반복 (입력받은 개수만큼):
        현재 숫자 + 1
        숫자 출력
        만약 현재 숫자가 31이면:
            "사용자 패배" 출력
            main으로 돌아가 게임 종료

종료


## 3. 핵심 기능 2: 컴퓨터 필승법 적용 로직
시작
    목표: 현재 숫자를 4의 배수(4, 8, 12, ..., 28)로 맞추기

    컴퓨터가 말할 개수 = (4 - (현재 숫자 % 4))
    단, 값이 0이면 1을 선택 (기본 안전값)

    반복 (컴퓨터가 선택한 개수만큼):
        현재 숫자 + 1
        숫자 출력
        만약 현재 숫자가 31이면:
            "컴퓨터 패배" 출력
            main으로 돌아가 게임 종료

종료
