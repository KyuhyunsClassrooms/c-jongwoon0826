# 1. 프로그램 설계서 (5점)
## 1. 프로젝트 주제
베스킨라빈스 31 게임 만들기
## 2. 핵심 기능 목록
1. 플레이어가 +1, +2, +3 중 하나를 선택하여 숫자를 말할 수 있도록 하기

2. 말한 숫자가 31을 넘지 않도록 제한하기

3. 컴퓨터가 베스킨라빈스 31 필승법(4의 배수 전략) 을 사용하여 번호를 말하게 하기

4. 31을 말한 사람이 패배하도록 결과 표시

5. 게임 종료 후 결과 출력

## 3. 데이터 구조 설계

**방법 1: 배열 사용 (기본)**
// 베스킨라빈스 숫자를 최대 31개 저장

int count = 0;  // 지금까지 말한 숫자 개수(=마지막 숫자를 알기 위해 사용)
